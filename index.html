Decoder, Encoder, Multiplexer & Demultiplexer

  • Decoder: mengubah kode biner n-bit menjadi 2ⁿ jalur output. Hanya satu output aktif pada satu waktu. Contoh: decoder 2→4, decoder 3→8.
  • Encoder: kebalikan decoder. Mengubah 2ⁿ input menjadi kode biner n-bit. Asumsi hanya satu input aktif. Priority encoder menangani beberapa input aktif dengan memprioritaskan yang tertinggi.
  • Multiplexer (MUX): memilih 1 dari beberapa jalur data input lalu meneruskannya ke 1 output. Pemilihan dikontrol select lines. MUX 4→1 butuh 2 select lines (2² = 4).
  • Demultiplexer (DEMUX): kebalikan MUX. Meneruskan 1 input ke salah satu dari beberapa output berdasarkan select lines. Sering disebut data distributor.

2

Half Adder & Full Adder

  • Half Adder: menjumlahkan 2 bit (A, B). Output: Sum = A ⊕ B, Carry = A · B. Tidak bisa menerima carry dari bit sebelumnya.
  • Full Adder: menjumlahkan 3 bit (A, B, Cᵢₙ). Output: Sum = A ⊕ B ⊕ Cᵢₙ, Cout = (A·B) + (B·Cᵢₙ) + (A·Cᵢₙ). Dapat dirangkai berantai (ripple carry adder).
  • 1 Full Adder = 2 Half Adder + 1 gerbang OR.
  • Untuk menjumlahkan bilangan n-bit diperlukan n buah Full Adder yang dirangkai seri.

3

ADC & DAC

  • ADC (Analog-to-Digital Converter): mengubah sinyal analog kontinu menjadi nilai biner digital. Proses: Sampling → Quantization → Encoding.
  • DAC (Digital-to-Analog Converter): kebalikan ADC. Mengubah kode biner menjadi tegangan analog. Digunakan pada audio player, penggerak motor, display.
  • Resolusi: ditentukan jumlah bit. n-bit → 2ⁿ level. Contoh: 8-bit = 256 level, 10-bit = 1024 level, 12-bit = 4096 level.
  • Teorema Nyquist: frekuensi sampling harus ≥ 2 × frekuensi maksimum sinyal (fₛ ≥ 2·fₘₐₓ). Jika dilanggar → aliasing (distorsi frekuensi).
  • Quantization error: selisih antara nilai analog asli dan nilai diskret terdekat. Semakin tinggi resolusi, semakin kecil error.

4

Register, JK Flip-Flop & Counter

  • Register: kumpulan flip-flop yang bekerja bersama untuk menyimpan atau menggeser data multi-bit.
  • SISO (Serial In Serial Out): data masuk 1 bit/clock, keluar 1 bit/clock. Berguna sebagai delay line atau buffer data serial.
  • SIPO (Serial In Parallel Out): data masuk serial, setelah n clock semua bit tersedia paralel. Mengubah serial → paralel.
  • PISO (Parallel In Serial Out): data dimuat paralel, keluar serial 1 bit/clock. Untuk transmisi data satu jalur.
  • PIPO (Parallel In Parallel Out): data masuk dan keluar paralel. Digunakan sebagai register penyimpanan sementara (buffer).

  • Counter Sinkron: semua FF menerima sinyal clock bersamaan. Tidak ada ripple delay, lebih cepat dan akurat. Cocok untuk frekuensi tinggi.
  • Counter Asinkron (Ripple Counter): hanya FF pertama yang mendapat clock luar; FF berikutnya dipicu oleh output FF sebelumnya. Lebih sederhana tapi ada propagation delay berantai.
